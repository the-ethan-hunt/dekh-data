# dekh-data


Dekh-data is a playground for data visualization in `Jupyter notebooks`. This includes [Tag clouds](https://en.wikipedia.org/wiki/Tag_cloud), [Treemapping](https://en.wikipedia.org/wiki/Treemapping) and much more!
